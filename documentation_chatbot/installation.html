ðŸ“‹ System Requirements

Prerequisites

Make sure you have the following installed on your system:

Backend Requirements
  • Python 3.8+ (recommended 3.11)
  • pip (Python package manager)
  • virtualenv or venv
  • Redis server (for caching and WebSocket)
Frontend Requirements
  • Node.js 16+ (recommended 18+)
  • npm or yarn package manager
  • Modern web browser
Port Usage
  • Port 8001: Django REST API
  • Port 8000: WebSocket (Daphne)
  • Port 8002: ChromaDB Server
  • Port 5173/5174: React Development Server
  • Port 6379: Redis Server

âš¡ Quick Start

One-Command Setup (macOS/Linux)

For quick setup, use our automated script:

# Clone the repository
git clone https://github.com/your-repo/full-chatbot-integration.git
cd full-chatbot-integration

# Run the setup script
chmod +x setup_macos.sh
./setup_macos.sh

What the script does:

  • Creates Python virtual environment
  • Installs all Python dependencies
  • Installs Node.js dependencies
  • Sets up database and runs migrations
  • Creates superuser account
  • Starts all services

ðŸ”§ Manual Installation

Step 1: Backend Setup (Django)

# Navigate to Django project
cd ChatBot_DRF_Api

# Create virtual environment
python -m venv chat
source chat/bin/activate  # On Windows: chat\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run database migrations
python manage.py migrate

# Create superuser
python manage.py createsuperuser

# Load initial data (optional)
python load_rag_data.py

Step 2: Frontend Setup (React)

# Navigate to React project
cd Chat-bot-react-main

# Install dependencies
npm install

# Start development server
npm run dev

Step 3: ChromaDB Setup

# Install ChromaDB dependencies (in Django virtual environment)
cd ChatBot_DRF_Api
source chat/bin/activate
pip install chromadb fastapi uvicorn

# Start ChromaDB server
python start_chromadb_simple.py

Step 4: Redis Setup

# Install Redis (macOS)
brew install redis
brew services start redis

# Install Redis (Ubuntu)
sudo apt update
sudo apt install redis-server
sudo systemctl start redis-server

# Verify Redis is running
redis-cli ping  # Should return "PONG"

ðŸŽ¯ Starting All Services

Service Order

Start services in this order for proper functionality:

1. Start Redis
redis-server
2. Start ChromaDB
cd ChatBot_DRF_Api
source chat/bin/activate
python start_chromadb_simple.py
3. Start Django API
cd ChatBot_DRF_Api
source chat/bin/activate
python manage.py runserver 127.0.0.1:8001
4. Start WebSocket
cd ChatBot_DRF_Api
source chat/bin/activate
daphne -p 8000 auth_system.asgi:application
5. Start React Frontend
cd Chat-bot-react-main
npm run dev
6. Serve HTML Pages
cd "New folder"
python -m http.server 3000
