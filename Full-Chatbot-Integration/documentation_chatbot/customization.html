🎨 Customization Guide

Customize chatbot appearance and behavior for your brand

🤖 Chatbot Widget Customization

Basic Widget Integration

Add the chatbot widget to any website with this simple script:

<!-- Basic Integration -->
<script src="http://localhost:8001/static/chatbot.js" data-company="SPE001"></script>

<!-- With Custom Configuration -->
<script 
  src="http://localhost:8001/static/chatbot.js" 
  data-company="SPE001"
  data-theme="dark"
  data-position="bottom-left"
  data-welcome-message="Hello! How can I help you today?"
  data-primary-color="#652d90"
  data-secondary-color="#3b82f6"
></script>

Widget Configuration Options

Attribute Description Default Options
data-company Company ID (required) - SPE001, NHS001, BOT001, etc.
data-theme Widget theme light light, dark, auto
data-position Widget position bottom-right bottom-right, bottom-left, top-right, top-left
data-primary-color Primary brand color #652d90 Any hex color
data-secondary-color Secondary color #3b82f6 Any hex color
data-welcome-message Initial greeting Hello! How can I help? Custom message
data-auto-open Auto-open widget false true, false
data-show-agent-info Show agent details true true, false

Company Branding

Add company-specific branding in the dashboard:

// src/components/CompanyBranding.tsx
interface CompanyBrandingProps {
  companyId: string;
}

export const CompanyBranding: React.FC<CompanyBrandingProps> = ({ companyId }) => {
  const brandingConfig = {
    SPE001: {
      logo: '/logos/spe001-logo.png',
      primaryColor: '#652d90',
      secondaryColor: '#3b82f6',
      companyName: 'Spell Consultancy'
    },
    NHS001: {
      logo: '/logos/nhs001-logo.png',
      primaryColor: '#1e40af',
      secondaryColor: '#059669',
      companyName: 'NHS Consultancy'
    }
  };

  const branding = brandingConfig[companyId] || brandingConfig.SPE001;

  return (
    <div className="company-branding">
      <img src={branding.logo} alt={branding.companyName} />
      <h1 style={{ color: branding.primaryColor }}>{branding.companyName}</h1>
    </div>
  );
};

Custom Dashboard Components

Create reusable components for consistent branding:

// src/components/CustomButton.tsx
interface CustomButtonProps {
  variant: 'primary' | 'secondary' | 'success' | 'danger';
  size: 'sm' | 'md' | 'lg';
  children: React.ReactNode;
  onClick?: () => void;
}

export const CustomButton: React.FC<CustomButtonProps> = ({
  variant,
  size,
  children,
  onClick
}) => {
  const baseClasses = 'font-medium rounded-lg transition-colors duration-200';
  
  const variantClasses = {
    primary: 'bg-purple-600 hover:bg-purple-700 text-white',
    secondary: 'bg-blue-600 hover:bg-blue-700 text-white',
    success: 'bg-green-600 hover:bg-green-700 text-white',
    danger: 'bg-red-600 hover:bg-red-700 text-white'
  };
  
  const sizeClasses = {
    sm: 'px-3 py-1.5 text-sm',
    md: 'px-4 py-2 text-base',
    lg: 'px-6 py-3 text-lg'
  };

  return (
    <button
      className={`${baseClasses} ${variantClasses[variant]} ${sizeClasses[size]}`}
      onClick={onClick}
    >
      {children}
    </button>
  );
};

COMPANY_RESPONSE_TEMPLATES = {

        'escalation': "I'll connect you with our expert team for detailed assistance.",

def get_response_template(company_id: str, template_type: str) -> str:
    """Get company-specific response template"""
    company_templates = COMPANY_RESPONSE_TEMPLATES.get(company_id, COMPANY_RESPONSE_TEMPLATES['SPE001'])
    return company_templates.get(template_type, "How can I help you today?")

Adding Language Support

Add multi-language support to your chatbot:

// Language configuration
const LANGUAGES = {
  en: {
    welcome: "Hello! How can I help you today?",
    typing: "Agent is typing...",
    send: "Send",
    file_upload: "Upload File",
    talk_to_agent: "Talk to Agent"
  },
  es: {
    welcome: "¡Hola! ¿Cómo puedo ayudarte hoy?",
    typing: "El agente está escribiendo...",
    send: "Enviar",
    file_upload: "Subir Archivo",
    talk_to_agent: "Hablar con Agente"
  },
  fr: {
    welcome: "Bonjour! Comment puis-je vous aider aujourd'hui?",
    typing: "L'agent tape...",
    send: "Envoyer",
    file_upload: "Télécharger le fichier",
    talk_to_agent: "Parler à un agent"
  }
};

// Auto-detect language or use data attribute
const userLanguage = document.documentElement.lang || 
                    navigator.language.substring(0, 2) || 
                    'en';

const t = (key) => LANGUAGES[userLanguage]?.[key] || LANGUAGES.en[key];
