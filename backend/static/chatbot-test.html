How It Works

The chatbot is loaded using a single line of JavaScript:

<script src="http://localhost:8002/static/chatbot.js" data-company="ZIL001"></script>

The script automatically:

  • Reads the data-company attribute
  • Fetches company-specific configuration from the API
  • Creates a responsive chat interface
  • Handles all the styling and positioning
